  • tr，指的是table row（表格行）。td，指的是table data cell（表格单元格）
  • th，指的是table header cell（表头单元格）。td，指的是table datacell（表行单元格）。
  • thead, tbody, 和 tfoot 元素默认不会影响表格的布局。不过，您可以使用 CSS 来为这些元素定义样式，从而改变表格的外观。
